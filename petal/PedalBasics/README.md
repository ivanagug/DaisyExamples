# PedalBasics

A basic pedal board with reverb and delay. More functions coming soon.

## Reverb
Bypass: **Switch 1**

Room Size: **Knob 1**

Send (dry/wet): **Knob 3**

## Delay
Bypass: **Switch 2**

Delay Time: **Knob 2**

Feedback: **Knob 4**

## Pitch Shifter
Semitones: **Encoder**

## Gain
Gain: **Knob 6**

## Install
Build: `make`
Install:
* Terminal - `make program-dfu`
* Online - https://electro-smith.github.io/Programmer/
